# 20. Influenza Whole-Genome Nextclade Datasets

Loculus uses official Nextclade datasets for every influenza segment rather
than the dashboard's HA-only influenza views. This allows ReVSeq ENA segment
records from the same sample to be represented as one multi-segment Loculus
entry while preserving segment-specific alignment, clade fields, and QC.
