# 12. Glossary

| Term | Meaning |
| --- | --- |
| CRAM | Compressed alignment file submitted as a downloadable Loculus file. |
| CRAI | Index file for a CRAM file. |
| ENA | European Nucleotide Archive. |
| LAPIS | Query API used by Loculus and downstream dashboards. |
| Nextclade | Existing preprocessing tool used for alignment, lineage/clade annotation, and mutation data. |
| RSV-A / RSV-B | RSV references configured as suborganisms in one Loculus RSV organism. |
| SILO | Sequence index used behind LAPIS. |
