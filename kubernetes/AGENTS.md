# Kubernetes Directory

## Formatting values.schema.json

After editing `kubernetes/loculus/values.schema.json`, run prettier to format it:

```bash
npx prettier@3.6.2 --write kubernetes/loculus/values.schema.json
```

## CDK8s

The Kubernetes manifests are generated by CDK8s (TypeScript) in `kubernetes/cdk8s/src/`. To format:

```bash
cd kubernetes/cdk8s && npm run format
```

To check formatting:

```bash
cd kubernetes/cdk8s && npm run check-format
```
